Better-cmd Documentation

Version 0.1.5-Stable
clear
Clears the screen by sending system call directly to terminal
ls
Lists directory contents in brief format (hidden files not shown)
ll
Lists directory contents with full details in vertical format (shows hidden files)
stat
Displays complete information about a specific file
cat
Displays content of text files
head
Similar to cat but only prints first 10 lines of file. Use -n to specify number of lines
tail
Similar to head but prints from bottom of the file instead of top
touch
Creates a new file with specified name
pwd
Prints current working directory path
basename
Takes a path and prints its last component
dirname
Takes a path and prints all components except the last one
now
Displays current date and time
sleep
Pauses execution for specified milliseconds
wc
Counts lines, words and characters. Supports pipe input
randnum
Generates random number between 0 and specified max (default 0-100)
randword
Generates random words (default 10 words)
env
Lists all environment variables
uptime
Displays system uptime
uniq
Processes text to remove duplicate lines. Supports both pipe and file input
man
Displays manual/help page for a command
paste
Outputs the current clipboard content
pure
Removes ANSI codes and emojis from input text
runtime
Executes a command and displays the execution time.
tr
Translates or removes the desired characters from the input.
trim
Removes extra spaces.
ps
Lists all running processes on your system, showing their PID and name.
kill
Terminates a process using its PID.
tac
Displays content of text files in reverse order (from last line to first)
hexdump
Display file contents in hexadecimal
